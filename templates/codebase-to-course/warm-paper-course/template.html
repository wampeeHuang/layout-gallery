代码 
async function fetchUser(id) {
  const res = await fetch(`/api/users/${id}`);
  if (!res.ok) throw new Error('Not found');
  return res.json();
}
逐行解释
声明异步函数 fetchUser，接收 id 参数
用 fetch API 请求用户数据，URL 动态拼接 id
响应非 OK 时抛出错误（触发调用方 catch）
解析 JSON 响应体并返回
